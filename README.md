 TITANIC DATA ANALYSIS 

 Overview

This project analyzes the Titanic dataset using Python to identify factors that influenced passenger survival.

 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

Analysis Performed

* Data inspection
* Missing value analysis
* Survival rate by gender
* Survival rate by passenger class
* Age distribution analysis
* Survival rate by age group

 Key Findings

* Female passengers had a higher survival rate than males.
* First-class passengers were more likely to survive.
* Children had relatively higher survival rates.
* Most passengers were adults aged 20–40.
* Some age values were missing.
